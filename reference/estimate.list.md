# Estimate method for lists

Estimate method for lists

## Usage

``` r
# S3 method for class 'list'
estimate(x, ...)
```

## Arguments

- x:

  list of model objects or `lvm` objects

- ...:

  Additional arguments to lower level functions

## Value

Object of class `estimate.list` (a list of `estimate` objects), or a
fitted multi-group model if input is a list of `lvm` objects.
