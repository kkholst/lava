# Appending `Surv` objects

`rbind` method for `Surv` objects

## Usage

``` r
# S3 method for class 'Surv'
rbind(...)
```

## Arguments

- ...:

  `Surv` objects

## Value

`Surv` object

## Author

Klaus K. Holst

## Examples

``` r
y <- yl <- yr <- rnorm(10)
yl[1:5] <- NA; yr[6:10] <- NA
S1 <- survival::Surv(yl,yr,type="interval2")
S2 <- survival::Surv(y,y>0,type="right")
S3 <- survival::Surv(y,y<0,type="left")
```
