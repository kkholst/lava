# Concatenation operator

For matrices a block-diagonal matrix is created. For all other data
types he operator is a wrapper of `paste`.

## Usage

``` r
x %++% y
```

## Arguments

- x:

  First object

- y:

  Second object of same class

## Details

Concatenation operator

## See also

`blockdiag`, [`paste`](https://rdrr.io/r/base/paste.html),
[`cat`](https://rdrr.io/r/base/cat.html),
